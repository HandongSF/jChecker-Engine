# jChecker-engine
jChecker는 Abstract Syntax Tree(ASF)를 이용하여 자바의 객체지향프로그래밍 개념을 자동으로 채점해주는 도구입니다. 현재까지의 프로그래밍 자동 채점 도구는 각 프로그래밍 언어가 가지는 고유의 특성을 온전히 반영하지 못하였지만, jChecker는 객체지향프로그래밍 개념을 반영한 자동 채점 도구로 개발되었습니다.
